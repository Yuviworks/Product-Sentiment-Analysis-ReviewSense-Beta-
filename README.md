# 🛍️ ReviewSense – Product Sentiment Analyzer

A modern web app that analyzes product reviews and predicts sentiment (Positive, Negative, Neutral) using Machine Learning.

---

## 🚀 Features

- Clean futuristic UI (Streamlit)
- Sentiment prediction using ML models
- Confidence score visualization
- Keyword extraction from reviews
- Example-based testing

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- TF-IDF Vectorization
- Streamlit
- Matplotlib

---

PSAP1/

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
